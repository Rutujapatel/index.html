I have been working on this project since last 3 months. I have given below the sketches I made and also information of automatic flute machines which are already made.





    Starting with the project

    This is the rough sketch I have made. Following are the components which I have used in my project:

    1. servo motor
    2. Arduino
    3. Blower Nozel
    4. Pressure Controller

    In this, the servo motors will control the holes of flute and I heve taken Arduino Mega to control servo motor. The servo will react according to the tune given. Even the nozel with the blower will react according to the pressure given. As in Indian flute half covered hole is also a svara so will have to adjust the servos accordingly. Indian flute is little different than the flute used in other countries so this will be a total different project and this will be first automatic Indian flute.

    The Blower Nozel and Pressure controller are yet to design properly. I am Going to 3D print the blower and the stand for flute.
