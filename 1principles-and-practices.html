Start with Git

  • Go to your git profile.
  • Click on now repository create it as index.html and clone it. This wil make an untitled folder on your desktop then name it as 1234
  • copy the repository link in terminal and click enter. This will connect your file to terminal
  • Now open index.html with sublimeand type basic html program of heading and paragraph. Save the changes.
  • Go to Terminal and write following programs
    1. cd desktop : this points the terminal to desktop
    2. cd 1234 : this points the terminal to folder in which our file is stored
    3. git status : This command is used to know if any change is made in the file or not. It also shows the list of liles which are changed
    4. git add -A : it adds all the changed files
    5. git status -m "abc" : here we use it to commit th echanges which we made
    6. git push : this command will push all the changes we made
